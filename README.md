# 🔮 Quizzical Trivia

Quizzical Trivia is a sleek, modern, dark-themed general knowledge trivia application designed to test your intellect across multiple categories. Built with high performance and smooth user interactivity in mind, the application dynamically fetches questions from the Open Trivia Database (OTDB API).

---

## 🚀 Features

*   **Dynamic Question Fetching:** Pulls random, real-time sets of 5 multi-choice questions spanning varying trivia categories.
*   **Difficulty Visual Indicators:** Features color-coded badges classifying questions clearly into `EASY`, `MEDIUM`, or `HARD`.
*   **Intuitive UI State Tracking:** 
    *   Answers turn a vibrant brand purple (`#5846E3`) upon selection.
    *   Instant validation highlights the correct answers in crisp green and wrong guesses in red when checking results.
*   **Confetti Celebration:** Reaching high scores triggers a colorful cascading canvas confetti celebration across the viewport.
*   **Infinite Replayability:** Seamlessly fetch a fresh batch of trivia seamlessly via the "Next questions" function without full page reloads.

## 🛠️ Tech Stack

*   **Frontend Library:** React (Functional Components & Hooks)

*   **Styling:** Custom CSS / Modern Flexbox & Grid
*   **API Ecosystem:** Open Trivia Database (OTDB)
